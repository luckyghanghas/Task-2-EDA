# Task 2 - Exploratory Data Analysis & Business Intelligence

This repository contains Task 2 deliverables for the 60-Day Data Analytics Internship. The work uses the cleaned sales transaction dataset from Task 1 to answer business questions with SQL and summarize insights through exploratory data analysis.

## Objective

Identify sales patterns, channel performance, product trends, customer behavior, and return risk using descriptive statistics, SQL queries, and simple dashboard visuals.

## Repository Structure

```text
data/
  sales_transactions_cleaned.csv
  sales_analysis.sqlite
dashboard/
  eda_charts.html
reports/
  eda_report.md
  sql_query_results.md
sql/
  business_questions.sql
README.md
```

## Business Questions Covered

1. What are the top products by revenue?
2. How is revenue trending month by month?
3. Which sales channel performs best?
4. Which region has the highest return rate?
5. Which campaign creates the strongest gross profit?
6. Which customer age group has the highest revenue per customer?

## Key Deliverables

- EDA report: `reports/eda_report.md`
- SQL queries: `sql/business_questions.sql`
- SQL results: `reports/sql_query_results.md`
- Cleaned dataset: `data/sales_transactions_cleaned.csv`
- SQLite database: `data/sales_analysis.sqlite`
- Dashboard mock-up charts: `dashboard/eda_charts.html`

## Summary Insight

The analysis shows that digital channels, especially Website and Mobile App, drive most revenue. Audio products need closer return-rate monitoring, while high-value customer age groups can be targeted for retention and loyalty campaigns.
